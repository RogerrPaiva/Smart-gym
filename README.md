# Smart Gym

Projeto desenvolvido em React + Vite com a proposta de uma academia inteligente, reunindo navegação entre páginas, apresentação da academia, planos, calculadora IMC e formulário de contato.

## Repositório

- [Smart Gym no GitHub](https://github.com/RogerrPaiva/Smart-gym)

## Funcionalidades

- Página Home com banner
- Informações importantes sobre a academia
- Cards dos planos mais comprados
- Página de planos
- Calculadora IMC
- Página de contato
- Footer
- Navegação com React Router

## Tecnologias

- React
- Vite
- CSS
- React Router DOM

## Rotas

- `/` Home
- `/planos` Planos
- `/imc` Calculadora IMC
- `/contato` Contato
- `/sobre` Sobre

## Estrutura do projeto

- `src/pages`: páginas principais da aplicação
- `src/components`: componentes reutilizáveis
- `src/styles`: arquivos de estilo CSS
- `src/assets`: imagens e recursos visuais
- `public`: arquivos públicos do projeto
