# Smart Gym

Projeto desenvolvido em React + Vite com proposta de uma academia inteligente, contendo páginas de navegação, apresentação da academia, planos, calculadora IMC e formulário de contato.

## Funcionalidades

- Página Home com banner
- Informações importantes sobre a academia
- Cards de planos
- Página de planos
- Calculadora IMC
- Página de contato
- Footer
- Navegação com React Router

## Tecnologias

- React
- Vite
- CSS
- React Router DOM
